# AVL-Trees
Simple trees and AVL implemented using C++ 

## **Run the code**

```bash
  git clone [link](https://github.com/c0rRupT9/AVL-Trees.git)
  && cd AVL-Trees/src
```
To compile the code
```bash
C++ main.cpp -o trees.exe
```
To run enter
```bash
trees
```
or for PS

```bash
./trees
```
